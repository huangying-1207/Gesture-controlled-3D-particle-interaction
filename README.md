# Gesture-controlled-3D-particle-interaction
The main functions are as follows: 1. 3D particle-based Christmas tree 2. Defined gesture action (fist): Gather the tree 3. Defined gesture action (open hand): Disperse into particles 4. Defined gesture action (pinching): Enlarge the image 5. Defined gesture action (peace sign): Switch images 6. Supports clearing and re-uploading images 7. etc.
